```shell
#查询userinfo索引库中所有的文档数据，根据age字段进行倒序输出
GET /userinfo/_search
{
  "query": { "match_all": {} },
  "sort":[
    {"age":{"order":"desc"}}
  ]
}
```

如果需要根据多个字段排序，则可以使用下面的语句：

```shell
GET /索引名称/_search
{
  "query": { "match_all": {} },
   "sort":[
    {"field1":{"order":"desc"}},
    {"field2":{"order":"asc"}}
    ]
}
```

以上语句表示先根据field1字段进行倒序排序，再根据field2字段进行顺序排序。

要注意的是，排序字段不可以是text等特殊类型，一般是整数类型和keyword类型。
