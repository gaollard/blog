尽管动态映射很有用，但是为了满足业务的各种需求，有时还是需要为索引显式创建映射关系。语法如下：

```json
#创建名为 `myindex007` 的索引库并为其创建映射关系
PUT /myindex007
{
  "mappings": {
     "properties": {
         "name":{
            "type": "text"
         },
         "age":{
           "type": "long"
         },
          "actiontime":
          {
            "type": "date",
            "format":"yyyy-MM-dd HH:mm:ss" 
          },
          "describe":
          {
            "type": "keyword"
          }
     }
  } 
}
```
