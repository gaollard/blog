## 1. 手机靓号-连续相同数字

```javascript
var str = '13249065444';

// 判断是否为手机号
var regLen = /^1[0-9]{10}$/;

// 判断是否包含连续3个相同的数字
var reg = /^.*(\d)\1\1(.*)$/;
var reg = /^.*(\d){3}$/;

console.log(regLen.test(str)); // true
console.log(str.match(reg)); // res[0]: 4
```
